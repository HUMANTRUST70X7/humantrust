# HumanTrust MVP - Estrutura de Pastas e Uso

1. Coloque a pasta 'humantrust' no seu computador.
2. Abra Vercel e crie um novo projeto.
3. Faça upload de toda a pasta 'humantrust'.
4. Teste:
   - index.html → Editor minimalista, gravação, análise e PDF.
   - verify.html → Verificação de certificados via QR Code ou código.
5. Scripts:
   - assets/01_script.js → Gravação + Pyodide
   - assets/02_pix.js → Pagamento via PIX
   - assets/03_styles.css → Layout clean
   - assets/04_generateManualPDF.js → PDF final
6. Dados:
   - data/01_certificates.json → Certificados válidos
7. Após testar, configure monetização e continue a auditoria digital e impressa.
